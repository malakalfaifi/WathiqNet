How It Works
  1. Drop log files into the watch folder
  2. The watcher detects new files and changes automatically
  3. Logs are parsed and analyzed by the detection engine
  4. Alerts are generated for suspicious activity
  5. Real-time updates are pushed to all connected browsers
Supported Log Formats
CSV Files (.csv)
timestamp,source_ip,destination_ip,source_port,destination_port,protocol,packet_size,flags,action,device_id
